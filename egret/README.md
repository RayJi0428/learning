## 基礎程式
### 網頁基礎
開發使用TypeScript，執行使用JavaScript
### 變數
``` typescript
let v0:boolean = true;              //真假值
let v1:number = 123;                //數字
let v2:string = "abc";              //字串
let v3:any = {"name":"a","value":3};//複雜物件
let v4:number[] = [1,2,3];          //陣列
```
### 函式
| 分類      | 內容           |
| --------- | -------------- |
| public    | 對外全開       |
| protected | 僅對繼承者開放 |
| private   | 私有           |
